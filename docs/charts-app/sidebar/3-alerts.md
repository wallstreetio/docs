# Alerts

## Overview

The Alerts widget lets the user create and manage price alerts for symbols, review triggered alerts in Alert Logs, and control how alert notifications are delivered.

## Guides

### Create a price alert

1. Click the "+" icon
1. Follow the prompts

### Create a price alert (alternate)

* See [Plot](../chart-area/2-plot.md#access-shortcut-actions).
* See [Doji Screener](../toolbar/1-doji-screener.md#set-price-alerts).

### Filter and sort alerts

1. In the top right of the Sidebar Display Area, click the three dots
1. From the dropdown, select filters and sorts as desired

### Delete visible alerts

1. In the top right of the Sidebar Display Area, click the three dots
1. Click "Delete Visible Alerts".

### Edit, clone, or delete an alert 
1. To the right of the alert, click the three dots
1. From the dropdown select "Edit", "Clone", or "Delete" as desired. 
1. Follow the prompts.

## Reference

### Alert conditions

Price alerts fire when a symbol's price is "Greater than" or "Less than" a set price. The user can set an expiration using "Tomorrow", "1 Week", "1 Month", or a custom date, and optionally add an alert name (max 40 characters).

### Notification options

When creating an alert, the user can enable:

* "Notify in app" — push notification in the mobile app
* "Show pop-up" — pop-up notification in the app
* "Play Sound" — sound with the pop-up notification
* "Send email" — email notification
