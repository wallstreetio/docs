# Watchlists

## Overview

Watchlists allow the user to organize symbols the user cares about, so the user can pull up the right chart fast and keep the user's focus where it belongs.

## Guides 

### Create a watchlist
1. Click the three-dot icon at the top-right
1. Click "create watchlist"
1. Edit window will open. From the edit window the user can: 
  *  Rename watchlist
  *  Search for and Add symbols


### Edit a watchlist
1. From watchlist selector, select desired watchlist
1. Click the plus sign
1. Edit window will open. From the edit window the user can: 
  *  Rename watchlist
  *  Add symbols
  *  Add color tags to symbols already in watchlist
  *  Rearrange symbols by dragging icon on the left
  *  Remove symbols

### Create watchlist section
1. Load custom watchlist
1. Click the three-dot icon
1. Click "Add section divider"
1. The new section divider will appear at the bottom. 

### Edit watchlist section
 * To add a symbols, drag the symbol under the section divider
 * To add multiple symbols, drag the section divider to the top of those symbols
