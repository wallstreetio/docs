# Strategy Builder

## Overview

The Strategy Builder is a powerful tool to create strategies. As the user is creating the strategy, they can see exactly how it would perform on Daily, Weekly, and Monthly candle looking back 1 to 20 years. The specifics of how to use it are best learned by opening it [in app](https://app.wallstreet.io/chart/AAPL?symbol=AAPL&signal=1&workspace=backtest&strategies=discover).
