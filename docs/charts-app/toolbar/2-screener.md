# Screener (Beta)

## Overview

The Screener is an improved Doji Screener. It is designed surface only the most relevant information for a specific stock while still allowing the user some free to explore other criterion that may be useful for evaluating a stock.

## Guides 

### Show or hide table columns

1. On the far right select "Show/Hide Columns" 
1. In the popout, select or un-select columns as desired 
1. Click back in the Screener area to hide popout

### Add a filter

1. Select "+ Filter"  
1. From the popout, select a filter 
1. Filter will populate in the Filter Area
1. Adjust filter using dropdowns 
1. Click checkbox on left of filter apply or disable filter 

### Create new saved filter 

1. Have at least one filter active
1. Click "Save"
1. Follow the prompt
