# Overview
The Chart app has all the tools a technical analysis trader could need to discover new strategies, evaluate those new strategies, and be alerted when it's time to trade.

## Components
The chart is made up of three components, each one holding similar things. 

* Chart Area: charting tools and the plot.
* Toolbar: bespoke tools for discovering and evaluating opportunities often used with the chart, but sometimes on their own. 
* Sidebar: Watchlists, Price Alerts, Chatroom

For more, navigate through the main menu, which is either visible on the left side, or folded up in the top left.
