# Plot

## Overview

The Plot displays symbol information as a time series.

## Guides 

### Change the symbol

1. In the upper left click the symbol
2. Search for and select symbol. 
3. The selected symbol will populate in the plot. 

### Change the symbol (alternate)

1. In Charting Tools, on left click the magnifying glass icon
2. Search for and select symbol. 
3. The selected symbol will populate in the plot.

### Change interval
1. In upper left click the current interval
2. From dropdown select desired interval
3. The selected interval will populate in the plot.

### Access shortcut actions 
1. Right-click on the Plot
1. Select shortcut action. Available shortcut options: 

    * "Reset Chart View"
    * "Add Price Alert" 
    * "Add to Watchlist"
    * "Remove Drawings"
    * "Remove Studies" 
    * "Remove Chart"

1. Follow the prompt

### Edit a drawing
1. Right-click on drawing
1. Use dropdown to edit desired aspects. 

### Delete a drawing 
1. Right-click on drawing 
1. Click "Delete Drawing"

### Delete a drawing (alternate)
1. Left click on drawing
1. Press delete button on the keyboard


## Reference

### Candle intervals 
The candle interval is how much time each candle on the chart represents. Available intervals: 

* 1 minute (1m) 
* 2 minute (2m) 
* 5 minute (5m)
* 10 minute (10m)
* 15 minute (15m)
* 30 minute (30m)
* 1 hour (1H)
* 4 hour (4H)
* 1 day (1D)
* 1 week (1W)
* 1 month (1M) 

### Shortcut actions 
To view Shortcut actions, right-click on the Plot. The "Remove Chart" option is only relevant if multiple charts are displayed. It removes the selected chart.
