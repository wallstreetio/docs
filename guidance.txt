General
* Don't use you, use the user. 
* Don't repeat yourself multiple times in different sections. Use "For more see link.", and link to the best section for the information to live. 
* When referring to specific components, use text from the app, quoted it, and write the text as it appears in app, eg click on "ALL STOCKS" or ensure on "Current Stock" is selected.
* When referring to a labelled icon, write text as it appears but not quoted eg Drawing tools

For Guides sections: 
* Only add a guide if multiple steps are involved.
* Don't add guide steps for filling out prompts.
* don't repeat steps between guides. If you find yourself repeating steps a lot, then add a guide for just that action. 
* The guide subsections should be singular unless specifically plural, eg  Add a stock to watch list 


For FAQ sections
* Only add an FAQ if the question has been asked at least twice. 
* Don't add them in anticipation. If it's general information that might be helpful, then add it to the reference section

Simple update workflow for Leslie
1. Go to https://wallstreetio.github.io/docs/ 
2. Open a file in the docs folder 
3. In the upper right, click on the pencil to edit.
4. Make changes. 
5. In the top right, click "Commit changes.."
6. Click through prompt. Commit is the same as saving.
