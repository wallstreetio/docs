## When to use 

* Use this skill when you need to get more information about wallstreet.io, or whenever you need to visit a url containing wallstreet.io

## Instructions 

* Use screenshots in docs/asserts/screenshots to get a general idea if you don't know where to start. All the sections are labelled.
* Use the playwright mcp server. It will already have auth that you need. 
* Go to the [wallstreet.io charts page](https://app.wallstreet.io/chart/AAPL?strategies=personal&signal=1&symbol=AAPL) unless it makes sense to start somewhere else. 
* Browse as desired.
