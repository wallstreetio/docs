# Check support tickets

## Instructions 

* Use intercom mcp to browse tickets that have been started and completed in the lat 7 days.
* Look for common questions and annotate them on a new scratch_suppport.md. A question is common if it is asked by at least 3 people. Annotated the conversation urls for each topic.
* Compare the common questions against topics covered in docs. 
* Propose additions as a list of numbers with conversation urls included 
* User will specify which numbers to implement
